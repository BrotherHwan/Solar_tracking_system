# Solar_tracking_system
## 프로젝트 목적
태양광패널이 태양광을 추적하도록하여 발전효율 향상.
## 사용 디바이스
라즈베리파이, 아두이노2개, 조도센서, 서보모터, 태양광패널, 인체감지센서, 부저, LCD패널, 블루투스 모듈, 와이파이 모듈
